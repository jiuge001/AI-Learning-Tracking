# 学习跟踪PWA系统 - 项目记忆

## 项目信息
- **项目名**：学习跟踪 - 齐芛&齐芃
- **路径**：E:\私人\孩子学习\AI学习任务\
- **技术栈**：纯HTML/CSS/JS + Chart.js + PWA
- **存储**：localStorage（主） + JSON导入导出（跨设备同步）
- **同步方案**：坚果云同步data目录下的JSON文件

- **学科**：语文、数学、英语

## 关键设计决策
1. 不使用钉钉/飞书/微信小程序（需要服务器和审核）
2. PWA可安装到手机桌面，Service Worker支持离线
3. 数据通过localStorage存储，导出JSON文件通过坚果云同步
4. 拍照上传使用 `<input capture="environment">` 调用手机相机
5. 练习题生成引擎内置于app.js，数学题支持分数运算等

## 用户偏好
- 手机使用频率远大于电脑
- 异地办公/出差场景需要同步
- 试卷拍照→OCR→错题分析（当前手动录入，OCR为增强功能）
- 家长完整共管（任务分配+讨论+提醒）
